# TV Show Notification App
It will send notification before airtime. ~ 15 min.


## Built with
```
- react
- redux
- redux-saga
- reselect
- redux-toolbox
- firebase
- cloud functions
- FCM
```

used AWS lambda & CloudWatch for cron job.

#### Data source : http://www.tvmaze.com/api
